*[ACL]: 액세스 제어 목록(Access Control List)
*[ALB]: Application Load Balancer
*[ANFW]: AWS Network Firewall
*[ARN]: Amazon 리소스 이름(Amazon Resource Name)
*[AZ]: 가용 영역(Availability Zone)
*[BGP]: 경계 게이트웨이 프로토콜(Border Gateway Protocol)
*[BW]: 대역폭(Bandwidth)
*[CDN]: 콘텐츠 전송 네트워크(Content Delivery Network)
*[CGW]: 고객 게이트웨이(Customer Gateway)
*[CIDR]: 클래스 없는 도메인 간 라우팅(Classless Inter-Domain Routing)
*[CLB]: Classic Load Balancer
*[DDoS]: 분산 서비스 거부(Distributed Denial of Service)
*[DHCP]: 동적 호스트 구성 프로토콜(Dynamic Host Configuration Protocol)
*[DNS]: 도메인 이름 시스템(Domain Name System)
*[DX]: AWS Direct Connect
*[EC2]: Elastic Compute Cloud
*[ELB]: Elastic Load Balancer
*[GWLB]: Gateway Load Balancer
*[IAM]: 자격 증명 및 액세스 관리(Identity and Access Management)
*[ICMP]: 인터넷 제어 메시지 프로토콜(Internet Control Message Protocol)
*[IPAM]: IP 주소 관리자(IP Address Manager)
*[IPv4]: 인터넷 프로토콜 버전 4(Internet Protocol version 4)
*[IPv6]: 인터넷 프로토콜 버전 6(Internet Protocol version 6)
*[MSS]: 최대 세그먼트 크기(Maximum Segment Size)
*[MTU]: 최대 전송 단위(Maximum Transmission Unit)
*[NACL]: 네트워크 액세스 제어 목록(Network Access Control List)
*[NAT]: 네트워크 주소 변환(Network Address Translation)
*[NLB]: Network Load Balancer
*[PPS]: 초당 패킷 수(Packets Per Second)
*[QoS]: 서비스 품질(Quality of Service)
*[RDS]: Relational Database Service
*[S3]: Simple Storage Service
*[SD-WAN]: 소프트웨어 정의 광역 네트워크(Software-Defined Wide Area Network)
*[SG]: 보안 그룹(Security Group)
*[SSL]: 보안 소켓 계층(Secure Sockets Layer)
*[TCP]: 전송 제어 프로토콜(Transmission Control Protocol)
*[TGW]: Transit Gateway
*[TLS]: 전송 계층 보안(Transport Layer Security)
*[TTL]: 유효 기간(Time To Live)
*[UDP]: 사용자 데이터그램 프로토콜(User Datagram Protocol)
*[VGW]: 가상 프라이빗 게이트웨이(Virtual Private Gateway)
*[VPC]: 가상 프라이빗 클라우드(Virtual Private Cloud)
*[VPN]: 가상 프라이빗 네트워크(Virtual Private Network)
*[WAF]: 웹 애플리케이션 방화벽(Web Application Firewall)
