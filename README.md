# AWS Networking Best Practices — Korean Edition (KO/EN)

A bilingual (Korean / English) site for **AWS networking best practices**. It mirrors and
translates [aws/aws-networking-best-practices](https://github.com/aws/aws-networking-best-practices)
into Korean, and additionally curates **AWS What's New** and **Networking blog** updates —
automatically filtered for networking topics and summarized in both languages.

🔗 **Site:** https://aws-samples.github.io/sample-aws-networking-best-practices-ko/

> ⚠️ The Korean text is an unofficial machine translation. Always refer to the English original for authoritative content.

## What you'll find

- **Networking best practices** — the full guide (foundation, connectivity, application networking, security, observability), in Korean and English.
- **News** — networking-relevant AWS What's New announcements and Networking & Content Delivery blog posts, automatically collected and summarized (KO/EN).
- **Bilingual** — switch languages anytime from the header (Korean at `/`, English at `/en/`).

Regenerating the translations and news requires AWS Bedrock access — see [`deploy/README.md`](deploy/README.md).
